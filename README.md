## 📖 Overview

My Favorite Movie is a web application that lets users browse, search, and curate a personal list of favorite films. It demonstrates vanilla JavaScript skills including DOM manipulation, local storage, and dynamic content rendering.

## ✨ Features

- 🎥 **Movie Discovery** — Browse and search for movies
- ❤️ **Favorites List** — Save and manage your personal movie collection
- 💾 **Local Storage** — Favorites persist across browser sessions
- 📱 **Responsive Design** — Works on desktop and mobile

## 🚀 Getting Started

```bash
git clone https://github.com/Rafik-Gharbi/my-favorite-movie.git
cd my-favorite-movie
# Open index.html in your browser
```

## 🛠️ Tech Stack
- Vanilla JavaScript
- HTML5 / CSS3
- Local Storage API
